# Глосарій термінів - Довідник дорожніх знаків

## Ключові терміни

### Дорожній знак (Road Sign)
- **Визначення**: Стандартизоване графічне зображення, що встановлює певний режим руху
- **Контекст**: Каталог знаків
- **Атрибути**: назва, категорія, опис, зображення

### Категорія знака (Sign Category)
- **Визначення**: Група дорожніх знаків з однаковим призначенням
- **Приклади**: Заборонні, Попереджувальні, Інформаційні, Пріоритету
- **Контекст**: Каталог знаків

### Користувач (User)
- **Визначення**: Особа, що взаємодіє з системою
- **Типи**: Гість, Адміністратор
- **Контекст**: Користувачі та доступ

### Адміністратор (Administrator)
- **Визначення**: Користувач з правами управління контентом
- **Права**: додавання, редагування, видалення знаків
- **Контекст**: Управління контентом

### Каталог (Catalog)
- **Визначення**: Систематизований перелік дорожніх знаків
- **Функції**: пошук, фільтрація, перегляд
- **Контекст**: Каталог знаків
