# Bounded Contexts - Контексти домену

## Контекст "Каталог знаків" (Core Domain)
- **Відповідає за**: перелік знаків, пошук, фільтрацію, категоризацію
- **Сутності**: RoadSign, SignCategory
- **Взаємодії**: 
  - Отримує дані про користувача з "Користувачі та доступ"
  - Надає API для перегляду знаків

## Контекст "Користувачі та доступ" (Supporting Domain)
- **Відповідає за**: аутентифікацію, авторизацію, ролі користувачів
- **Сутності**: User, Role
- **Взаємодії**: Надає інформацію про користувача іншим контекстам

## Контекст "Управління контентом" (Supporting Domain)
- **Відповідає за**: додавання, редагування, видалення дорожніх знаків
- **Сутності**: Admin, ContentModeration
- **Взаємодії**: Використовує "Каталог знаків" для змін контенту
